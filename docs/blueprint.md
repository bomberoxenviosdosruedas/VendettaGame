# **App Name**: Vendetta Game

## Core Features:

- Inicio de Sesión Seguro: Permite a los usuarios iniciar sesión de forma segura con correo electrónico y contraseña utilizando Supabase Auth y SSR.
- Registro: Permite a los nuevos usuarios crear cuentas.
- Olvidé mi Contraseña: Permite a los usuarios restablecer sus contraseñas si las olvidan.
- Inicio de Sesión Social: Ofrece opciones de inicio de sesión social para una autenticación más rápida.
- Soporte de Idioma Español: Todo el texto y los mensajes de error orientados al usuario están en español.

## Style Guidelines:

- Carmesí profundo (#8B0000) para representar la intensidad y el enfoque estratégico, como corresponde al tema del proyecto.
- Gris muy oscuro (#121212) para mantener un ambiente arenoso, inmersivo y serio. Esto establece un telón de fondo ideal para mostrar las métricas del proyecto.
- Un color beige (#F5F5DC) para elementos interactivos y resaltados, manteniendo la coherencia con el color primario pero con mayor brillo para el contraste.
- Fuente del cuerpo y del título: 'Inter', sans-serif, que proporciona una estética limpia y moderna a la vez que preserva la legibilidad para un gran público de usuarios.
- Diseño similar a la imagen adjunta, utilizando buenas prácticas de programación (código definido en pequeños componentes específicos para cada página).
